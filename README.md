

PARA FINS DE TESTES: https://romaflix.top


Este repositório contém apenas o **frontend** (HTML, CSS, JavaScript puro) — sem backend, sem dados, sem configurações de infraestrutura.

## O que está incluído

- `index.html` — Página principal (SPA)
- `style.css` — Estilos combinados
- `css/` — Estilos modulares (base, catálogo, modal, player, páginas, responsivo)
- `js/` — JavaScript modular do frontend

## O que NÃO está incluído

- Backend / Worker
- Banco de dados / SQL
- Configurações de deploy
- Chaves de API
- Dados de catálogo

## Tecnologias

- HTML5, CSS3, JavaScript vanilla (sem frameworks)
- TMDB API (para imagens de posters)
- HLS.js (para reprodução de vídeo adaptativo)
